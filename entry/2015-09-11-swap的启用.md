> 开启交换分区的步骤如下：

```
# 创建所需大小的交换分区文件,下面的命令就创建了2G的交换分区文件，其中bs的代表块大小，默认单位为字节
dd if=/dev/zero of=/path/to/swap bs=1024 count=2048000

# 将上面的文件创建交换分区文件
mkswap /path/to/swap

# 启用交换分区
swapon /path/to/swap

# 至此交换分区启用成功，可在free命令中查看其使用情况

```
