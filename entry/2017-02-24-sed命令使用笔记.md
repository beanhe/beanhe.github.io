### sed用法
- 在指定行插入指定字符，控制项`i`代表在匹配行之前插入，控制项`a`代表在匹配行之后插入，例如：
 ```
 # 在首行加入"---"
 sed -i '1i ---' <file_name>
 
 # 在含有"title:"字符的行前插入"---"
 sed -i '/title:/i ---' <file_name>
 
 # 在含有"tags:"字符的行后插入"aaa"
 sed -i '/tags:/a aaa' <file_name>
 ```
 
- 对匹配行下一行进行操作，如删除含有`pattern:`字符的下一行内容：

```
sed -i '/pattern:/{n;d}' <file_name>
```
